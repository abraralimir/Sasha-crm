# Sasha-crm

Sasha-crm is an intelligent, AI-powered Customer Relationship Management (CRM) platform designed to streamline sales, task management, and team collaboration. It integrates cutting-edge AI automation with a sleek, intuitive interface to empower your team, enhance workflow efficiency, and amplify results.

## Core Features

- **Platform-Aware AI Assistant:** Engage with Sasha, an AI that has a deep, real-time understanding of your entire CRM. Ask complex questions and get instant, data-driven answers.
- **Intelligent Automation:** Automate repetitive tasks from lead creation to task assignment, predict lead ROI, and assess risks to free your team for strategic growth.
- **Instant Team Sync:** A centralized hub for real-time communication, including integrated group chat with file sharing and task creation capabilities.
- **Integrated HD Conferencing:** Seamlessly transition from chat to a face-to-face video conference directly within the platform.
- **Uncompromising Security:** Built on an enterprise-grade security model with a mandatory verification gate and granular data access rules to protect your data at every layer.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS with ShadCN UI components
- **Database & Auth:** Firebase (Firestore & Authentication)
- **Generative AI:** Google Genkit
- **Video Conferencing:** VideoSDK
